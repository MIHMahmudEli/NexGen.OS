# NexGen.OS 🚀

NexGen.OS is a premium, minimalist developer bookmark hub designed for maximum productivity and aesthetic excellence. It features a sleek glassmorphism interface, dynamic background animations, and a powerful theming engine.

![NexGen OS Preview](https://github.com/MIHMahmudEli/NexGen.OS/raw/main/preview.png)

## ✨ Features

- **💎 Elite Design**: Modern glassmorphism UI with vibrant mesh background animations.
- **🎨 Multi-Theme Engine**: Switch between 5 professional themes (Midnight, Emerald, Solar, Ocean, Sunset) with persistent memory.
- **🖱️ Drag & Drop**: Intuitively reorder your bookmarks with smooth, staggered animations.
- **🔍 Command & Control**: Integrated search and URL synchronization bar for rapid access.
- **📂 Categorization**: Automatic categorization of bookmarks with a clean, minimalist sidebar.
- **📱 Responsive**: Fully optimized for various screen sizes.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with advanced animations, transitions, and CSS variables for theming.
- **JavaScript**: Core logic for bookmark management, drag-and-drop, and localStorage persistence.
- **Lucide Icons**: Crisp, professional iconography.

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/MIHMahmudEli/NexGen.OS.git
   ```
2. Open `index.html` in your favorite browser, or serve it locally:
   ```bash
   npx http-server
   ```
